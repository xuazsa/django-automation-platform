# Django 自动化平台

一个基于 Django 开发的企业级自动化运维平台。

## 功能特性

- **自动化平台**：定时任务管理、Cron 表达式、执行日志
- **交换机管理**：设备台账、SSH 命令执行、配置备份
- **Python 在线执行器**：在线编写代码、支持 input()、Matplotlib 图表
- **系统监控**：CPU/内存/磁盘监控、服务状态、历史趋势图
- **告警规则**：自定义阈值告警
- **博客系统**：文章发布、分类标签
- **后台管理**：Django Admin 完整管理

## 技术栈

- Django 4.2
- SQLite / MySQL
- psutil、matplotlib、netmiko
